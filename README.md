# OVGE
Open source vulkan game engine

# Environment setup
1. Install MSYS2
2. Open MSYS2 console as admin and install
   a. pacman -S mingw-w64-ucrt-x86_64-gcc
   b. pacman -S mingw-w64-ucrt-x86_64-cmake
   c. pacman -S mingw-w64-ucrt-x86_64-make
   d. pacman -S mingw-w64-ucrt-x86_64-gdb
   e. pacman -S mingw-w64-ucrt-x86_64-glfw
   f. pacman -S mingw-w64-ucrt-x86_64-glm
   g. pacman -S mingw-w64-ucrt-x86_64-vulkan-devel
